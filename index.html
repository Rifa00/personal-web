    App Contact
    App Contact sederhana ini berbasis CLI yang dibuat menggunkan node.js di dukung dengan beberapa package Manager yaitu yargs, chalk , dsb. dan App CLI ini hanya berjalan di node.js saja. proyek ini dibuat pada tanggal 5 juli 2024. fiturnya sangat sederhana hanya dapat tambah kontak, hapus kontak, update kontak, list kontak, dtail kontak dan juga search.
    • cara penggunaan:
    • 1.Tambah kontak -> node app add --name="nama kontak" --email="contoh@gmal.com".
    • selain menambahkan nama dan email bisa juga menambahkan no hp(--no="083..") dan deskripsi (--desc="Lorem5 ") namun yang pokok adalah nama dan email.
    • 2.update kontak -> node app up --email="contoh@gmail.com" --newName="Rifa" dan sesuaikan saja apa yang mau di update.
    • 3.list semua kontak -> node app list.
    • 4.cari kontak -> node app src --name="nama kontak"
    • 5.dtail kontak -> node app dtl --email="contoh@gmail.com"
    • 6.hapus kontak -> node app dlt --email="contoh@gmail.com"
